* linux 基础命令
** 常用命令
   在基于字符界面的 linux 操作系统下有一组使用频度最高的命令。
   |---------+-------------------------------------------------+----------------------------------------+-----------------------|
   | 命令    | 作用                                            | 参数                                   | 示例                  |
   |---------+-------------------------------------------------+----------------------------------------+-----------------------|
   | ls      | list directory content                          |                                        |                       |
   |---------+-------------------------------------------------+----------------------------------------+-----------------------|
   | cd      | change current diretory                         |                                        |                       |
   |---------+-------------------------------------------------+----------------------------------------+-----------------------|
   | chown   | change file owner and group                     |                                        |                       |
   |---------+-------------------------------------------------+----------------------------------------+-----------------------|
   | chmod   | change file mode bites                          |                                        |                       |
   |---------+-------------------------------------------------+----------------------------------------+-----------------------|
   | ps      | report a snapshot of the current processes      |                                        | ps -ejH               |
   |---------+-------------------------------------------------+----------------------------------------+-----------------------|
   | ln      | make links between files                        |                                        |                       |
   |---------+-------------------------------------------------+----------------------------------------+-----------------------|
   | history | show history command                            |                                        |                       |
   |---------+-------------------------------------------------+----------------------------------------+-----------------------|
   | tar     |                                                 |                                        | tar -xvf source.tar   |
   |         |                                                 |                                        | tar -cvf dest.tar *.* |
   |---------+-------------------------------------------------+----------------------------------------+-----------------------|
   | unzip   |                                                 |                                        | unzip source.zip      |
   |---------+-------------------------------------------------+----------------------------------------+-----------------------|
   | zip     |                                                 |                                        | zip *.*               |
   |---------+-------------------------------------------------+----------------------------------------+-----------------------|
   | alias   | set the other name for command                  |                                        |                       |
   |---------+-------------------------------------------------+----------------------------------------+-----------------------|
   | su      | run a command with substitute user and group ID |                                        |                       |
   |---------+-------------------------------------------------+----------------------------------------+-----------------------|
   | sudo    | execute a command as another user               |                                        |                       |
   |---------+-------------------------------------------------+----------------------------------------+-----------------------|
   | find    | search command                                  | -name filename #查找名为filename的文件 | find . -name tt       |
   |         |                                                 | -type b/d/c/p/l/f #根据文件类型查找    |                       |
   |         |                                                 | -user username #按文件属主来查找       |                       |
   |         |                                                 | -perm 755 #按执行权限来查找            |                       |
   |         |                                                 | -prune ~ #忽略某个目录                 |                       |
   |         |                                                 | -maxdepth 1 #指定搜索目录深度          |                       |
   |         |                                                 | ...                                    |                       |
   |---------+-------------------------------------------------+----------------------------------------+-----------------------|
   | grep    | print lines matching a pattern                  | see the man page                       | grep -nH "tt" -r ./   |
   |---------+-------------------------------------------------+----------------------------------------+-----------------------|
   | file    | determine file type                             |                                        |                       |
   |---------+-------------------------------------------------+----------------------------------------+-----------------------|
