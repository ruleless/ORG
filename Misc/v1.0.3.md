# 个人项目

## 快速端口转发工具

  - 项目地址：[https://github.com/ruleless/fasttun](https://github.com/ruleless/fasttun)
  - 代码线上浏览地址：[http://src.ruleless.net/source/xref/fasttun/](http://src.ruleless.net/source/xref/fasttun/)
  - 功能：通过端口转发，为C/S架构的软件提供高速的网络数据转发服务
  - 原理：通过一套比TCP流速更快的ARQ实现(KCP)来提升网络数据传输速度。

## 基于Nginx的扩充功能模块：快速端口转发

  - 项目地址：[https://github.com/ruleless/nginx/tree/master/src/port_map](https://github.com/ruleless/nginx/tree/master/src/port_map)
  - 简介：功能基本上同上述C++编写的快速端口转发工具，但基于Nginx的设计使其具有更高、更灵活的可配置性。

## 通用网络数据转发代理

  - 项目地址：[https://github.com/ruleless/proxy](https://github.com/ruleless/proxy)
  - 功能：基于TCP的通用反向代理服务框架
  - 描述：在基于分布式的服务部署方式中，我们一般不会把所有服务都暴露在公网环境中。
          通常，我们会提供一到多台专门用于桥接用户和实际业务服的"连接服"(或称反向代理服务器)。
		  "连接服"的核心功能有两条：1、对外，接收用户请求，并转发给业务服；
		  2、对内，接收业务服响应，并转发给用户。
		  上述逻辑可以抽象成通用逻辑，本项目即致力于此！
